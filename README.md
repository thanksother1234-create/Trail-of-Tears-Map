# Trail of Tears Map

This project is an interactive historical atlas for the Trail of Tears, designed as a classroom-friendly visual exploration of the forced removal routes of the Cherokee, Choctaw, Chickasaw, Creek, and Seminole nations.

## What this project includes

- An interactive map-based experience for exploring removal routes
- Route cards and highlighted points for each nation
- Historical context for each route, including dates and descriptions
- A simple Vite + React + Leaflet setup for local development

## Project structure

- [index.html](index.html) — Main HTML entry point for the app
- [src/main.tsx](src/main.tsx) — React entry file that mounts the application
- [script.js](script.js) — Route data and interactive behavior for the map and info card
- [styles.css](styles.css) — Styling for the map, cards, and interface

## Getting started

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

## Notes

This repository uses Vite with React and Leaflet to present the historical content as an interactive web experience. The content is focused on educational storytelling rather than a full GIS platform.
